Dijkstra's Algorithm

Find the shortest path between nodes in a graph. Perfect for GPS navigation and network routing.

Time: O(V²) Space: O(V)

Bubble Sort

Simple comparison-based sorting algorithm. Repeatedly steps through the list and swaps adjacent elements.

Time: O(n²) Space: O(1)

Selection Sort

In-place comparison sorting algorithm. Divides input into sorted and unsorted regions.

Time: O(n²) Space: O(1)

Insertion Sort

Efficient for small data sets. Builds final sorted array one item at a time with comparisons.

Time: O(n²) Space: O(1)
